# Excel Sales Report Generator

A Python CLI tool that analyzes sales data from an Excel file and generates a summary report.

## Features

* Calculate total sales
* Find top customer
* Summarize sales by product
* Clean CLI report output

## Technologies

* Python
* Pandas
* Excel

## Example Output

```
=== SALES REPORT ===

Total Sales: 62500

Top Customer:
A 32000

Sales by Product
Laptop : 60000
Keyboard : 2000
Mouse : 500
```

## How to Run

Install dependencies

```
pip install pandas openpyxl
```

Run the script

```
python report.py
```
